Data types

Python has many different built-in Data Types all-ready to use immeadiately after installation. The following however, are the main ones:

  • Integers
  • Strings
  • Lists
  • Dictionaries
  • Booleans

Integers are numbers which don't have decimal points, raging from -infinity to +infinity

Strings are concatenations of unicode characters

Lists are mutable sequences

Dictionaries are mutable unordered collections

True and False values are the keywords which comprise the boolean data type

Declaring variables

To declare a variable it's only necessary to write its name and equate it to its intended value like so:

example_var = 55
Arithmetic operators

Atithmetic operators are special characters used to perform calculations. They are the following

  • Addition + : returns the sum of two expressions
  • Substraction - : returns the difference of two expressions
  • Multiplication * : returns the product of two expressions
  • Power ** : returns the value of a numeric expression raised to a specified power
  • Division / : returns the quotient of two expressions
  • Floor division // : returns the integral part of the quotient
  • Modulud % : returns the decimal part (remainder) of the quotient

For loops

The for statement is used to iterate over the elements of a sequence (such as a string, tuple or list) or other iterable object

Syntax:
